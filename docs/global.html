# LayoutPrivate() → {JSX.Element}

Componente de diseño para rutas privadas.

Este componente se asegura de que solo los usuarios autenticados puedan acceder a las rutas privadas. Si el usuario no está autenticado (sin token), lo redirige a la página de inicio de sesión. Si el usuario está autenticado, renderiza el Header, el contenido de la ruta (Outlet), y el Footer.
